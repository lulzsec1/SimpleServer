# SimpleServer
Create Simple Files Sharing Server

# Installation
~# git clone https://github.com/lulzsec1/SimpleServer.git
# Setup
# Run as root
   ~# ./install.sh
# Usage

## STEP 1 :
   ~# ./simpleserver.sh
